# Claude Code e Antigravity na Prática

Site de divulgação e ementa do curso **"Claude Code e Antigravity na Prática: IA de Agentes para Servidores da Justiça"**.

## Estrutura

```
├── index.html          → site de divulgação (página de vendas)
├── ementa/
│   └── index.html      → ementa completa do curso (7 páginas, pronta para PDF)
├── assets/
│   └── wesley-photo-clean.png
└── netlify.toml        → headers de segurança e cache
```

## Sobre o curso

- **Carga horária:** 10 horas · 3 encontros ao vivo
- **Plataforma:** Google Meet
- **Público:** Servidores da Justiça Eleitoral, do Trabalho e demais ramos
- **Instrutor:** Wesley Wagner de Brito Silva — Técnico Judiciário · TRE-PB

## Deploy

Hospedado no **Netlify**. A cada `git push` o site é atualizado automaticamente.
